# Calibre Read History

Simple client-side web app to extract Calibre-Web read status from local app.db and optional metadata.db.

Files:
- calibre-read-history-app.html — main app

Usage:
Open the HTML file in a modern browser, select your `app.db` and optionally `metadata.db`, then click "Auswertung starten".

About:
This app is a client-side browser tool for reading Calibre-Web status data from `app.db` and optional `metadata.db` files.
